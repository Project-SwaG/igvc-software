Welcome to our Lidar GUI!
Most of the functions and menu options are self explanatory.

Menus

  • File
    Capture
    - This saves a snapshot of the current lidar frame.
    Exit
    - Closes the application
  • Lidar
    NAV200
    - Connect to the NAV200 lidar hardware.
    Load File
    - Loads a file saved from the Capture function into a SimulatedLidar instance.
    Default
    - Loads a SimulatedLidar instance with the default pattern.
  • View
    Points
    - Draws data as points in space.
    Lines
    - Draws rays connecting the origin (center of lidar device) to the point of data in space.
    This mimics the invisible infrared beams used by the lidar.

Interacting with data

You can zoom the data using the horizontal slider across the top of the data view. To move the origin of the display, simply click and drag the data around the screen.
